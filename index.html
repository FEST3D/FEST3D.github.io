FEST3D Documentation

Finite-volume Explicit Structured 3-Dimensional solver

What is FEST3D?

FEST3D is a computational fluid dynamics software written in Fortran 90 and Python for solving Navier-Stokes equations on a structured grid with state of the art finite-volume numerical method. This software is the proprietary of CFD Lab headed by Dr. Santanu Ghosh in the Department of Aerospace Engineering at the Indian Institute of Technology (IIT) Madras, Chennai

Language

Use of a compiler language, Fortran 90, with an object ordient scripting language, Python, provides speed and effiecenty in the workflow.

Modular

FEST3D maintains high modularity in the code sturture to keep door open for future changes, addition, and improvments.

Parallel

FEST3D is a distributed-memory based multiblock code, wherein the computational domain is divided in to multiple hexahedral blocks that communicate with each other using MPI messages.
